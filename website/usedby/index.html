Gigo in use

Please see below a list of websites which were generated with Gigo. We provided the urls of their source-codes which can be useful to learn through examples.

Kubegres

Kubegres is an open source Kubernetes operator which creates a cluster of PostgreSql databases. Kubegres uses Gigo to generates its official website.

Source code of their website using Gigo: https://github.com/reactive-tech/kubegres-website

Url of their website: https://www.kubegres.io

Reactive Tech Limited
